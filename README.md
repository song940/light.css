## light.css [![light.css](https://img.shields.io/npm/v/light.css.svg)](https://npmjs.org/light.css)

> A lightweight CSS framework

### Installation

```bash
$ npm install light.css --save
```

### Example

```html
<link rel="stylesheet" href="//unpkg.com/light.css/dist/light.css" />
```
